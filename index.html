Employee Management System

The Employee Management System is a responsive and interactive web application built using HTML, CSS, and JavaScript, focused on managing employee-related information through a user-friendly interface. This frontend-only system allows users to perform essential operations such as adding, editing, deleting, and viewing employee records directly from the browser. It features separate dashboards for HR and employees, search and filter functionality, and the ability to export data to CSV format.

PGLife

The PGLife Project is a web-based application developed as part of the Internshala Web Development Training Program, aimed at helping students and working professionals find Paying Guest (PG) accommodations across various cities and states in India. Built using HTML, CSS, Bootstrap, JavaScript, PHP, and MySQL, the platform features a clean, responsive UI where users can browse PG listings based on city, view property details such as rent, amenities, and location, and even register or log in to save preferences.
